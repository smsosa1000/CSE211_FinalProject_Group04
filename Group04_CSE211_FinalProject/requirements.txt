EVENTSX - PROJECT TECHNOLOGY STACK

This project utilizes modern web technologies to deliver a dynamic and interactive user experience.

CORE TECHNOLOGIES:
- HTML5: Semantic structure and form elements.
- CSS3: Advanced styling, Flexbox/Grid layouts, and custom animations.
- JavaScript (ES6+): Class-based logic, asynchronous Fetch API, and DOM manipulation.
- PHP: Server-side authentication and registration management.

LIBRARIES & TOOLS:
- FontAwesome: Scalable vector icons.
- Google Fonts: Modern typography (Inter/Roboto).
- Parallax Effects: Custom interactive visual components.

ARCHITECTURE:
- Responsive Design: Optimized for both desktop and mobile devices.
- Session-Based Auth: Secure server-side state management without LocalStorage.
